# BurnSense

BurnSense is a web-based application that detects emotional burnout
using mobile usage behavior patterns while ensuring user privacy.

## Features
- Burnout score calculation
- Usage pattern analysis
- Visual dashboard
- Health recommendations

## Tech Stack
- React
- Node.js
- Express
